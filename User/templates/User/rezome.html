Backend

با django کارمیکنم

- میتونم model های مختلف بسته به نیازتون بسازم مثل model پروفایل کاربرها (userprofile)که اکثر سایت ها یا appها بهش احتیاج دارن و ازش استفاده میکنن.

کاربراتون میتون ویژگی های مختلفی داشته باشند.

میتونم سایتتون رو روی هاست هایی که خدمات پایتونی میدن deploy کنم.

میتونم با cpanel کار کنم.

- با mysql آشنایی دارم (البته خود جنگو دستورات لازم رو به دیتابیس میفرسته ولی اگه اشتباهی حین migration ها یا migrate رخ بده میتونم مستقیما از mysql درستش کنم )و از اون بعنوان دیتابیس بک اند استفاده میکنم.

- میتونم برای ساییتون ایمیل و درگاه پرداخت قرار بدم و جفتشون به جنگو وصل خواهند بود.

- از git استفاده میکنم.

تجربه کار با django rest framework رو دارم

با django rest framework داده های سایتتون رو به فرمت json تبدیل میکنم و برنامه نویس اندروید یا سمت clientتون میتونه اونا رو در گوشی یا صفحه خاصی نمایش بده و یا کار خاصی روشون انجام بده.

- میتونم دسترسی به api رو محدود کنم به خواسته ی شما

Frontend

-از jquery استفاده میکنم.

-از ajax یا axios برای فرستادن request ها استفاده میکنم.
